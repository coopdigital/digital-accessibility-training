On this page are examples of the:

  • Foundations
  • Elements
  • Layout
  • Components
  • Patterns

You can use these in prototypes by opening the _includes folder, navigating to the code snippet then copying and pasting it.

Go to the Experience Library to find out how to apply the foundations, elements, components and patterns to your designs.

You can pass data around pages by copying the form example and following the instructions in the readme file to create your own interactive prototypes.
